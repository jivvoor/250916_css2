성장하는 백엔드 개발자
최지우입니다.

저는 눈에 보이지 않는 서버, 데이터베이스, 그리고 애플리케이션 로직을 설계하고 구현하며 서비스의 심장을 만드는 일에 큰 보람을 느낍니다. 안정적인 데이터 처리와 효율적인 API 설계를 통해 사용자가 매끄러운 경험을 할 수 있도록 돕는 것을 목표로 합니다.
'성장하는' 개발자라는 말처럼, 저는 현재에 안주하지 않고 끊임없이 새로운 기술을 탐구하고 동료들과 지식을 공유하며 함께 발전하는 것을 중요하게 생각합니다. 견고한 아키텍처와 깔끔한 코드로 신뢰할 수 있는 서비스를 만드는 개발자가 되겠습니다.

여행 추천 프로젝트

사용자 취향과 여행 데이터를 기반으로 맞춤형 여행지를 추천하는 API를 개발했습니다. Spring Boot와 JPA를 활용하여 효율적인 데이터 처리 로직을 구현했습니다.

재능 매칭 프로젝트

재능을 가진 사람과 필요한 사람을 연결하는 플랫폼의 백엔드 시스템을 구축했습니다. 실시간 매칭 알고리즘과 안전한 결제 연동 기능 개발을 담당했습니다.

AWS 인프라 프로젝트

AWS를 활용하여 확장 가능하고 안정적인 서비스 인프라를 구축했습니다. EC2, RDS, S3 등 주요 서비스를 연동하고, CI/CD 파이프라인을 설계하여 배포 자동화를 구현했습니다.
